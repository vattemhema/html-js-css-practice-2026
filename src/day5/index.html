The content between the opening and closing tags becomes the clickable link, known as anchor text. Basic Syntax The fundamental syntax for an anchor tag involves the href attribute, which specifies the destination URL. html Link Text Key Attributes Anchor tags use several attributes to define link behavior and destination: href: The most important attribute; it specifies the URL or path the link points to. This can be an absolute URL (e.g., https://www.example.com) or a relative URL (e.g., /about.html). target: Specifies where to open the linked document. _self: (Default) Opens the link in the same window or tab. _blank: Opens the link in a new window or tab. _parent / _top: Used with framesets (less common today). title: Provides extra information about the link, which is often shown as a tooltip when a user hovers over the link. rel: Specifies the relationship between the current document and the linked document (e.g., nofollow, noopener, noreferrer). download: Instructs the browser to download the linked URL as a file instead of navigating to it. A suggested filename can be specified as the attribute's value. Common Use Cases & Examples Use Case Example Code Description External website Visit Google Links to a different website.
