# Git Commands Log

## Deleting a Branch

1. **List branches** to confirm current and available branches:
   ```bash
   git branch
   ```

2. **Check status** to ensure working tree is clean:
   ```bash
   git status
   ```

3. **Switch to main** and **delete the branch**:
   ```bash
   git checkout main
   git branch -d hema/14jan2026-taskid-html-css
   ```
